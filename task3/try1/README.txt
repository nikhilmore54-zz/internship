# Prerequisites
-Install and configure a database server such as MySQL.

# Running the application
1. Copy the folder.
2. Edit the .env file and make the changes as per your database configuration.
3. Run the folowing in your terminal
    ```
    $ php artisan serve
    ```
    This should launch the application as a web server which would be accessble on http://localhost:8000
4. Now, open a web browser and enter "http://localhost:8000" to open the application.
5. Enter name and email and click register
6. you should see the web-page with Thank-you message.
